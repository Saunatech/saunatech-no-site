# Legal Source Documents

These Markdown files are source copies from the live BookSauna legal document set.

`booksauna.app` remains the live source of truth. The root HTML files in this repo are static, versioned compliance copies for reviewers and domain verification on `www.saunatech.no`.

When BookSauna legal documents change:

1. Copy the updated Markdown source files into this directory.
2. Update the corresponding root HTML pages.
3. Keep the version, effective date, last updated date, live URL, and contact metadata visible on each HTML page.
4. Keep internal links in the HTML pages pointed to local `.html` routes where appropriate.
